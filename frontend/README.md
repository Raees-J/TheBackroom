## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the landing page.

### Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Inter (sans-serif), JetBrains Mono (monospace)

## Components

- `Navbar` - Glassmorphic navigation with mobile menu
- `Hero` - Main hero section with animated badge and CTAs
- `DemoSection` - Interactive WhatsApp + Database demo
- `BentoGrid` - Feature showcase in bento box layout
- `FAQ` - Accordion-style FAQ section
- `Footer` - Terminal-inspired footer with newsletter

## Color Palette

- **Primary Green:** #3ECF8E

- **Darker Background:** #0F0F0F
- **Gray:** #2A2A2A
